# Emoji-Grill
Every needed another fun way to communicate with your friends? Introducing Emoji Grill. This encrypts your messages into emojis. Yeah emojis. You can encrypt and decrypt your messages. Comes with a password system so you and your friend only can access the messages.

## Features
- Encrypting and Decrypting all in the same spot
- Password System for secure commuication
- Chaos Kitchen Themed Website

## More Details
This is just a fun hobby project I made. Use this to have some fun with your friends. Dont actually use this for a secure communication method lol. Have fun.
